# Overview

Voitris is a classic tetris game with voice-controlled commands.

#

## Functionality
* Users will speak following command words to controll the game.

## Command Words
* up = "UP"
* down = "DOWN"
* turn = "TURN"

## Extra Commands
* switch = "SWITCH"
